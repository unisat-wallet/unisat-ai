# UniSat Skill for Claude Code

UniSat blockchain data integration - query Bitcoin blocks, transactions, BRC20, Runes, and Ordinals.

## Quick Install

```bash
curl -sSL https://raw.githubusercontent.com/unisat-wallet/unisat-ai/main/install-skill.sh | bash
```

## What's Included

- **16+ MCP Tools**: Blocks, Transactions, Addresses, BRC20, Runes, Ordinals, Marketplace
- **Auto Configuration**: Sets up MCP server and skill in one command
- **API Key**: Get your key at https://developer.unisat.io

## Usage

After installation, use in Claude Code:

```
@unisat What's the current Bitcoin block height?
@unisat Check ORDI top holders
@unisat Get current network fees
@unisat bc1q... balance
```
