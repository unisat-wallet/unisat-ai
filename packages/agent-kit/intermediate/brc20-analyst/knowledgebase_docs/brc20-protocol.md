# BRC20 协议文档

## 概述

BRC-20 是一个在比特币区块链上实现代币标准的实验性协议，由 Twitter 用户 @domodata 在 2023 年 3 月提出。它利用 Ordinals 铭文功能来创建、铸造和转移代币。

## 技术原理

### Ordinals 铭文

BRC-20 建立在 Ordinals 协议之上，Ordinals 允许用户在 satoshis（sats）上铭刻内容，这些被铭刻的 sats 称为"铭文"（inscriptions）。

### JSON 格式

BRC-20 使用 JSON 格式在铭文中存储操作指令：

```json
{
  "p": "brc-20",
  "op": "deploy",
  "tick": "ordi",
  "max": "21000000",
  "lim": "1000"
}
```

## 操作类型

### 1. Deploy（部署）

部署新的 BRC-20 代币。

```json
{
  "p": "brc-20",
  "op": "deploy",
  "tick": "代币ticker",
  "max": "总供应量",
  "lim": "每次铸造限制"
}
```

**参数说明：**
- `p`: 协议版本，固定为 "brc-20"
- `op`: 操作类型，"deploy" 表示部署
- `tick`: 代币标识符，4 个字符
- `max`: 最大供应量
- `lim`: 每次铸造的上限

### 2. Mint（铸造）

铸造已部署的 BRC-20 代币。

```json
{
  "p": "brc-20",
  "op": "mint",
  "tick": "ordi",
  "amt": "1000"
}
```

**参数说明：**
- `p`: 协议版本
- `op`: 操作类型，"mint" 表示铸造
- `tick`: 要铸造的代币标识符
- `amt`: 铸造数量

### 3. Transfer（转账）

转移 BRC-20 代币到另一个地址。

```json
{
  "p": "brc-20",
  "op": "transfer",
  "tick": "ordi",
  "amt": "100"
}
```

**参数说明：**
- `p`: 协议版本
- `op`: 操作类型，"transfer" 表示转账
- `tick`: 要转移的代币标识符
- `amt`: 转账数量

## 流程说明

### 铸造流程

1. 部署者发送 deploy 铭文，创建新代币
2. 用户发送 mint 铭文进行铸造
3. 每个 mint 铭文铸造不超过 lim 指定的数量
4. 当总铸造量达到 max 时，铸造结束

### 转账流程

1. 发送者发送 transfer 铭文，指定接收地址和数量
2. transfer 铭文被记录在比特币区块链上
3. 接收者通过解析铭文获得代币余额

## 风险与限制

### 技术风险

1. **实验性协议**: BRC-20 是实验性的，可能存在未知漏洞
2. **UTXO 管理**: 复杂的 UTXO 管理可能导致资金丢失
3. **区块链确认**: 交易需要比特币区块确认，速度较慢

### 市场风险

1. **价格波动**: BRC-20 代币价格波动极大
2. **流动性风险**: 部分代币流动性不足
3. **诈骗风险**: 存在大量仿盘和虚假项目

### 操作风险

1. **无效操作**: 错误的操作可能导致代币丢失
2. **交易费用**: 比特币网络费用可能很高
3. **确认延迟**: 网络拥堵时确认时间很长

## 数据指标

### 关键指标

- **总供应量 (max)**: 代币的最大供应数量
- **已铸造 (minted)**: 当前已铸造的数量
- **进度 (progress)**: 铸造进度百分比
- **持有人数 (holders)**: 持有该代币的唯一地址数量
- **交易次数**: 该代币的总交易次数

### 市场指标

- **地板价**: 市场最低挂单价格
- **24h 交易量**: 过去 24 小时的交易量
- **市值**: 总供应量 × 当前价格
- **持仓集中度**: 前 N 大持有者的持仓占比

## 常见代币

### ORDI (ordi)

- 第一个 BRC-20 代币
- 总供应量: 21,000,000,000
- 2023年3月部署

### SATS (sats)

- 以 satoshis 命名的代币
- 总供应量: 2,100,000,000,000,000
- 高度分散

## 开发建议

### 查询代币信息

使用 UniSat API 查询 BRC-20 代币信息：
- 获取代币基本信息
- 查询地址余额
- 获取持有人列表
- 获取市场数据

### 风险评估

评估 BRC-20 代币时考虑：
- 持仓分散度（持有人数越多越分散）
- 交易活跃度（交易次数和交易量）
- 市值和流动性
- 项目背景和社区
